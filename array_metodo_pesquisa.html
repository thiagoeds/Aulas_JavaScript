Array - Método de Pesquisa

Nesta aula o nosso desafio e bem simples, nos vamos aprender como podemos pesquisar elementos dentro dos nossoas arrays.

A ideia é pesquisar o elemento para verificar se ele existe ou não e a partir disso tomar algum tipo de decisão dentro da lógica das nossas aplicações.

Vamos criar o array lista_frutas para implementarmos a Pesquisa, vamos recuperar o array e colocar algumas posições e um tipo de fruta para cada uma delas.

var lista_frutas = Array()
lista_frutas[0] = ' Banana'
lista_frutas[1] = ' Maçã'
lista_frutas[2] = ' Morango'
lista_frutas[3] = ' Uva'

Para realizar uma pesquisa dentro do array nos utilizamos o metodo indexof que é nativo de objetos do tipo array, para isso, basta recuperar o nosso array lista_frutas e executar o método indexof passando o elemento que nós queremos pesquisar, neste caso vamos pesquisar o Morango, para visusalizar esta informação, vamos dar um console.log dessa pesquisa, para ver nos navegador qual é o retorno desse método.

console.log(lista_frutas.indexOf('Morango'))

Para esta pesquisa trazer resultado, o elemento da pesquisa precisa esta da mesma que foi inserido no array (espaços, letras maiuscula e minusculas e caractere especial), caso contrario ira retornar -1.
