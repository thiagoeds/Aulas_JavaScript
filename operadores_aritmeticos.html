Operadores Aritméticos

O que são operadores aritméticos?

Operadores:

  • Adição (+) - Soma valores;
  • subtração (-) - Diferença entre valores;
  • Multiplicação (*) - Produto dos valores;
  • Divisão (/) - Quociente dos valores;
  • Módulo (%) - Resto existente de uma operação de divisão;
  • Incremento (++) - Pré/Pós incremento;
  • Decremento (--) - Pré/Pós decremento.

var num1 = 9

var num2 = 2

document.write('A soma entre ' + num1 + ' e ' + num2 + ' é: ' + ( num1 + num2 ) )

document.write('A subtração entre ' + num1 + ' e ' + num2 + ' é: ' + ( num1 - num2 ) )

document.write('A multiplicação entre ' + num1 + ' e ' + num2 + ' é: ' + ( num1 * num2 ) )

document.write('A divisão entre ' + num1 + ' e ' + num2 + ' é: ' + ( num1 / num2 ) )

document.write('O módulo de ' + num1 + ' e ' + num2 + ' é: ' + ( num1 % num2 ) + ' ou seja, é a sobra da divisão do numero inteiro, 9 dividido por 2 é igual 8 e sobra 1 para dividir, o módulo vai retornar este numero que sobra.' )

document.write('A incremento pré entre ' + num1 + ' é: ' + ( ++num1 ) + ' ele adiciona +1 a operação quando colocado ++ depois, outro ponto de atenção ele trouxe o resultado 10 porque ele vai pegar o ultimo valor apresentado no browser ' )

document.write('O resultado é: ' + num1 )

document.write('A incremento pos entre ' + num1 + ' é: ' + ( num1++ ) + ' ele adiciona +1 a operação quando colocado ++ depois, porém ele atualiza este valor apenas na proxima operação, outro ponto de atenção ele trouxe o resultado 10 porque ele vai pegar o ultimo valor apresentado no browser ' )

document.write('O resultado é: ' + num1 )

document.write('A decremento pré entre ' + num1 + ' é: ' + ( --num1 ) )

document.write('O resultado é: ' + num1 )

document.write('A decremento pos entre ' + num1 + ' é: ' + ( num1-- ) )

document.write('O resultado é: ' + num1 )
