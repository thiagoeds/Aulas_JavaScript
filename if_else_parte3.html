Operadores Lógicos

O que são Operadores lógicos?

Operadores:

  • E (&&) - Verdadeiro se todas as expressões forem Verdadeiras.
  • OU (||) - Verdadeiro se pelo menos umas das expressões for verdadeira.
  • Negação (!) - Inverte o resultado da expressão de comparação.

Vejamos o exemplos de && abaixo:

//true && true = true
if(2 == 2 && 3 >= 1) {
document.write('Verdadeiro') }

Segundo exemplo de &&:

//true && true && false = true
if(2 == 2 && 3 >= 1 && 'a' == 'b') {
document.write('Verdadeiro')
} else {
document.write('Falso')
}

Vejamos o exemplos de || abaixo:

//false && false && false = false
if(4 == 2 || 3 >= 5 || 'a' == 'b') {
document.write('Verdadeiro')
} else {
document.write('Falso')
}

Vejamos o exemplos de ! abaixo:

//false = false
if(!(4 == 2)) {
document.write('Verdadeiro')
} else {
document.write('Falso')
}

Segundo exemplo:

//true = false
if(!(5 >= 2)) {
document.write('Verdadeiro')
} else {
document.write('Falso')
}

Outro exemplo logico seria é na criação de regra para saber se o aluno foi aprovado ou reprovado a partir da nota em relação as faltas.

Para isso vamos usar o exercicio da aula anterior:

var nota = prompt('Digite a nota do aluno:')
var nota = prompt('Digite a quantidade de faltas:')

var media = 7
var faltas_maximas = 15

if (nota >= media && faltas <= faltas_maximas) {
