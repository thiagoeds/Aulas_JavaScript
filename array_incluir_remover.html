Array - métodos de inclusão e exclusão de elementos

Nesta aula vamos aprender como podemos incluir e excluir elementos dos nossos arrays porem de forma programatica, até então nos estavamos declarando os nossos agentes com valores pré estabelecidos, mas é natural que dentro da logica dos nossos codigos, exista necessidade de incluir ou remover elementos desses arrays, portanto, é importante compreender como que nós podemos fazer isso de modo programatico atravez dos nossos scripts

Vamos ver 4 elementos principais:

  1. push: Incluir elemento no final do array
  2. unshift: Incluir elemento no inicio do array
  3. pop: Excluir elemento no final do array
  4. shift: Excluir elemento no inicio do array
