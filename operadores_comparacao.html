Operadores de Comparação (condições)

Também conhecidos como operadores relacionais, servem para formar as expressões condicionais para o comando if
Operadores:

Igual( = = ) - Verifica se os valores comparados são Iguais;
Identico ( = = = ) - Verifica se os valores comparados são iguais e do mesmo tipo;
Diferente ( ! = ) - Verifica se os valores comparados são diferentes;
Não idêntico ( ! = = ) - Verifica se os valores comparados são diferentes e de tipos diferentes;
Menor ( < ) - Verifica se o valor da esquerda é menor que o valor da direita;
Maior ( > ) - Verifica se o valor da esquerda é maior que o valor da direita;
Menor igual ( < = ) - Verifica de o valor da esquerda é menor ou igual ao valor da direita;
Maior igual ( > = ) - Verifica de o valor da esquerda é maior ou igual ao valor da direita;
